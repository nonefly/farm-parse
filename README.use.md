# 农场土地信息解析
用于解析农场游戏土地信息，本软件用于交流学习，当前仅支持 Windows 用户。
## 使用说明
1. 首次使用：双击 `farm-parse.exe` 启动程序
2. 授权验证：打开浏览器访问 `http://127.0.0.1:8787/auth.html`，复制机器码给管理员授权
3. 下载并安装证书：点击「下载证书」按钮，双击下载的证书文件，按照向导安装到「受信任的根证书颁发机构」[详细步骤参考：证书安装步骤]
4. 开始解析：点击「开启解析」按钮，系统会自动配置代理
5. 打开游戏：进入好友农场，解析结果会自动显示在页面上
6. 结束解析：使用完成后点击「关闭解析」按钮恢复网络设置
## 证书安装步骤
1. 点击页面上的「下载证书」按钮
2. 双击下载的 `farm-parse-ca.cer` 文件
3. 点击「安装证书」
4. 选择「本地计算机」（或「当前用户」）
5. 选择「将所有证书放入下列存储」
6. 点击「浏览」，选择「受信任的根证书颁发机构」
7. 点击「确定」→「下一步」→「完成」
## 注意事项
- ！！！开启代理后，之前进入的农场土地信息不会被解析，需要重新进入游戏
- ！！！使用结束后请关闭代理，避免影响正常网络！
- 如遇到网络问题请检查代理是否关闭:「设置-网络与Internet-代理-手动设置代理」
- 本工具只解析农场土地信息，不会对请求做任何改动，或主动发起请求
- 证书只需安装一次，后续使用无需重复安装
