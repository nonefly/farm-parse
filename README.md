# 农场变异解析工具

用于解析农场游戏好友土地信息，自动拦截 WebSocket 流量并展示变异植物数据。

## 快速开始

### 开发模式运行

```powershell
npm install
npm start
```

打开浏览器访问：`http://127.0.0.1:8787/proxy.html`

### 打包成可执行文件

```powershell
npm install
npm run build:win
```

打包完成后，在 `dist` 目录找到 `farm-parse.exe`。

## 使用说明

### Windows 用户

1. **首次使用**：双击 `farm-parse.exe` 启动程序
2. **安装证书**：打开浏览器访问 `http://127.0.0.1:8787/auth.html`，先完成授权验证
3. **下载并安装证书**：点击「下载证书」按钮，双击下载的证书文件，按照向导安装到「受信任的根证书颁发机构」
4. **开启代理**：点击「开启代理」按钮，系统会自动配置代理
5. **打开游戏**：进入好友农场，解析结果会自动显示在页面上
6. **关闭代理**：使用完成后点击「关闭代理」按钮恢复网络设置

### 端口信息

- 展示页面：`8787`
- HTTP/HTTPS 代理：`8788`

## 证书安装步骤

1. 点击页面上的「下载证书」按钮
2. 双击下载的 `farm-parse-ca.cer` 文件
3. 点击「安装证书」
4. 选择「本地计算机」（或「当前用户」）
5. 选择「将所有证书放入下列存储」
6. 点击「浏览」，选择「受信任的根证书颁发机构」
7. 点击「确定」→「下一步」→「完成」

## 注意事项

- 证书只需安装一次，后续使用无需重复安装
- 使用结束后请关闭代理，避免影响正常网络
- 本工具只会解析 `VisitService.Enter` 方法的好友土地信息
