🌾 Farm Parse

农场游戏数据解析工具 - Vercel Serverless 部署版

⚠️ 重要说明

由于 Vercel Serverless 环境的限制，本部署版本仅提供植物数据和变异类型的查询 API，不包含代理服务器功能。如需完整的代理功能，请下载桌面客户端或在本地运行。

📡 可用 API

GET /api/index?action=plants
获取所有植物数据列表
GET /api/index?action=mutations
获取所有变异类型映射
GET /api/index?action=health
健康检查接口

💡 使用示例

在浏览器中直接访问：

/api/index?action=plants

或使用 JavaScript：

fetch('/api/index?action=plants')
  .then(res => res.json())
  .then(data => console.log(data));

📦 完整功能

如需使用完整的代理服务器功能，请：

  • 下载桌面客户端（Windows / macOS）
  • 或在本地运行：npm start
