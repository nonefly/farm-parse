成熟时间 / 好友作物

查看 SQLite 中记录的好友 GID、农作物、土地、成熟时间、倒计时、成熟可摘数量。视觉调度后续也会挂在这个页面，不再单独做一个重复入口。

进入成熟时间页面查看服务状态

启动方式

  1. 执行 npm start。
  2. 打开 http://127.0.0.1:8787/，从卡片进入各功能。
  3. 所有页面和 API 默认都从同一个前台端口访问。
内部仍会有一个隐藏解析服务端口，由启动器自动管理，平时不需要打开。
