抓包解析 / 变异查看

原主页面。开启系统代理、下载证书、解析农场 WebSocket 数据，展示好友农场和自己农场的土地、作物与变异信息。

进入解析页面 下载证书
⏰

成熟时间 / 好友作物

查看 SQLite 中记录的好友 GID、农作物、土地、成熟时间、倒计时与可摘状态。成熟时间服务默认运行在 8790 端口。

🧺

摘取调度 / 视觉队列

同一个成熟时间页面中包含摘取任务。规则是提前 5 分钟访问好友校准，最后 1 分钟连续点击收取坐标，完成后回访刷新。

调度说明

推荐启动顺序

  1. 启动原解析服务：npm start，然后从本页进入「抓包解析」。
  2. 另开终端启动成熟时间服务：npm run maturity，然后从本页进入「成熟时间 / 好友作物」。
  3. 后续视觉自动化只需要对接成熟时间服务的本地命令队列，不直接发送游戏协议请求。
如果你通过局域网 IP 访问本页，成熟时间链接会自动使用同一个主机名并切换到 8790 端口。
